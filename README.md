# Sauron
Face recognition door lock using AWS rekognition, RaspberryPi, RabbitMQ, paho

## 💻프로젝트 소개
---

"AWS Rekognition"서비스를 활용한 안면인식 도어락입니다.


## ⏱️개발 기간
-2019.06 ~ 2019.11 (창의설계프로젝트)

## 📄구현기능
- 도어락 원격 제어
- 실시간 스트리밍
- 사용자 추가
- 얼굴 인식 및 도어락 해제
